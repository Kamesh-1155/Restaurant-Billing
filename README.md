## 📌 Features

* Menu Management
* Order Processing
* Bill Generation
* Database Integration using SQLite
* CSV-based Menu Support
* Backend Logic Separation
* Simple and Beginner-Friendly Structure

## 📂 Project Structure

```bash
Restaurant-Billing/
│
├── app.py                # Main application interface
├── main.py               # Entry point of the project
├── backend.py            # Backend functionalities
├── db.py                 # Database connection and operations
├── menu.py               # Menu handling functions
├── order.py              # Order processing logic
├── menu.csv              # Menu items data
├── menu.jpg              # Menu image
├── restaurant.db         # SQLite database file
├── test_menu.py          # Testing menu functions
└── __pycache__/          # Compiled Python files
```

## 🛠️ Technologies Used

* Python
* SQLite3
* CSV Handling
